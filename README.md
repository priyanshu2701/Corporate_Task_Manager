#Corporate Task Management System:-

A role-based Corporate Task Management System designed to streamline task allocation, project tracking, and performance monitoring across multiple departments in an organization.

This system ensures clear hierarchy, controlled access, and smooth workflow between management and team members.

#User Roles & Permissions:-

#Super Admin:-

Full system access across all departments (Sales, Development, Graphics, Digital)

Create, assign, and manage projects and tasks

Promote, demote, or delete any user

Monitor and review submitted tasks and reports from all users

Complete authority over system operations

#Department Head:

Same task & project management powers as Super Admin

Restricted to their own department only

Can assign tasks and review submissions within their team

#Employee:

View assigned tasks and projects

Submit completed work and reports

Assign tasks to Interns

#Intern:

Access only assigned tasks

Submit task work for review

# Key Features:

Role-based authentication & authorization

Department-wise access control

Task & project assignment system

Hierarchical reporting structure

Secure user management (promote/demote/delete)

Centralized monitoring dashboard
